Singleshot Help

Hopefully Singleshot is pretty straightforward to use. The system is simple enough:

  1. Click the shortcut recorder, and type your desired shortcut.
  2. Select the desired language that your shortcut will switch the system to.
  3. There is no step 3.

Troubleshooting

If it seems like something's not working right, check the following things:

  • Is your desired language installed? Check your International settings in System Preferences.
  • Ensure "Enable access for assistive devices" is on in the Accessibility part of System Preferences.

Getting Help

Last but not least, if you think you've found a real bug, or have a feature request, send it to one of two places:

  • If you're the technical type, file it on our Github issue tracker.
  • If you're not a very technical person, or just have a general question, send an email to development@sunseaskyfactory.com.

Above all, thank you for using this program!
